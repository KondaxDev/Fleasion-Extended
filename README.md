# Fleasion+




An enhanced and community-driven fork of **Fleasion**, expanding the original project with a Python addon ecosystem, built-in utilities, automatic updates, and quality-of-life improvements.

Fleasion+ keeps everything that makes Fleasion great while introducing new features.

## ✨ Features

### Core Fleasion Features

* Asset replacement
* Cache scraper
* Cache viewer
* Mesh & animation preview
* Community presets
* Proxy-based asset interception
* Multi-platform support (Windows, macOS & Linux)

### New in Fleasion+

* 🔌 **Python Addon System**

  * Create your own addons using Python.
  * Extend Fleasion without modifying the source code.
  * Easy addon loading and management.

* ❤️ **Account Heal**

  * Built-in utilities to check if your Roblox account is banned, working, or experiencing issues.

* 👀 **Creator Watch**

  * Notifies you when a creator posts a new asset ID using Discord webhooks! (Experimental, not guaranteed to work.)

* 🔄 **Automatic Updates**

  * Check for new versions directly from the Settings menu.
  * Download the latest version from the GitHub repository without needing to download Fleasion again.

* ⚡ **Quality of Life Improvements**

  * Additional settings.

## Why Fleasion+?

Fleasion+ is designed to be an extension of the original Fleasion project rather than a replacement.

The goal is to provide an actively maintained fork with additional functionality and community-driven features while staying compatible with the original Fleasion experience.

## Installation

Download the latest version from the **Releases** page.

No Python installation is required for the standalone executable.

If you want to build from source, follow the same build instructions as the original Fleasion project.

## Addon System

One of the biggest additions in Fleasion+ is the Python Addon System.

Developers can create addons that integrate directly into Fleasion+, allowing the community to expand the application's functionality without modifying the core project.

Future addon examples include:

* Draw new colors on assets directly inside Fleasion+.
* "Replay" mode to restore your assets after making changes.
